# faber-taa
See results of the Meb Faber Tactical Asset Allocation (TAA) strategy on whatever ticker available on Yahoo!® Finance.

[yfinance](https://github.com/ranaroussi/yfinance) is used to query Yahoo!® Finance APIs.
